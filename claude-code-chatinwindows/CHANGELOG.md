### 新增
- ✨ 核心命令系统
  - 更新协议命令

  - 导出协议
  - 提交新 Tip
  - 整合 Tips（使用 AWS Bedrock）

- 🔧 配置管理
  - 仓库路径配置
  - 自动更新设置
  - AWS Bedrock 集成配置
  - 本地 LLM 配置（预留）

- 📦 Git 集成
  - 自动克隆仓库
  - 自动 pull 更新
  - 提交和推送 Tips
  - 仓库状态检查

- 🧠 智能协议管理
  - 协议解析和合并
  - 保留自定义内容
  - 协议完整性验证
  - 备份和恢复机制

- 💡 Tips 系统
  - Tips 列表管理
  - 待整合/已整合状态跟踪
  - 自动归档
  - AI 辅助整合

### 技术实现
- TypeScript 重构所有核心逻辑
- VS Code Extension API 集成
- simple-git 封装
- AWS SDK Bedrock Runtime 集成
- 完整的错误处理和恢复机制

- AWS 凭证配置需要手动触发
- 批量应用功能待实现

## 版本说明

### [版本号规则]
- **主版本号**: 不兼容的 API 变更
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### [变更类型]
- `新增`: 新功能
- `变更`: 现有功能的变更
- `废弃`: 即将移除的功能
- `移除`: 已移除的功能
- `修复`: 问题修复
- `安全`: 安全性相关的更新
