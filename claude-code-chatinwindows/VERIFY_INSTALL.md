# 验证 Awesome ClaudeMD Manager 安装

## ✅ 安装状态检查

您的插件已成功安装: `awesome-claudemd@2.1.1`

但您看到的 "Claude Code Chat v2.1.0" 是另一个扩展：`lkbaba.claude-code-chatui`

## 🚀 如何使用我们的插件

### 重要说明
- **我们的插件不是聊天窗口**
- **我们是 CLAUDE.md 协议管理工具**
- **提供独立的可视化管理面板**

### 打开方式

#### 方式 1: 状态栏（推荐）⭐
在 Cursor 窗口底部状态栏**右侧**，找到：
```
☁ ClaudeMD v2.1.1
```
点击它 → 打开主面板

#### 方式 2: 命令面板
1. 按 `Ctrl+Shift+P`
2. 输入: `Awesome ClaudeMD: 打开主面板`
3. 回车

#### 方式 3: 所有可用命令
按 `Ctrl+Shift+P` 后输入 "Awesome ClaudeMD" 可以看到所有命令：
- 打开主面板
- 更新协议
- 应用协议到当前项目
- 导出 CLAUDE.md
- 打开 Tips 管理
- 提交新 Tip

## 🔍 如何确认插件已安装

### 检查扩展列表
1. 打开扩展面板 (`Ctrl+Shift+X`)
2. 搜索 "awesome-claudemd" 或 "Awesome ClaudeMD"
3. 应该看到已安装的标记

### 检查状态栏
重启 Cursor 后，状态栏右侧应该出现：
```
☁ ClaudeMD v2.1.1
```

如果没有看到，说明插件未激活，请尝试：
1. 完全关闭 Cursor
2. 重新打开
3. 打开任意文件夹/项目

## 📸 界面预览

### 主面板界面
- 顶部显示: "🚀 Awesome ClaudeMD 管理器 v2.1.1"
- 有独立定位说明
- 显示当前协议状态
- 提供各种操作按钮

### 与 Claude Code Chat 的区别

| 功能 | Claude Code Chat | Awesome ClaudeMD |
|------|------------------|------------------|
| 界面 | 聊天窗口 | 管理面板 |
| 用途 | AI 对话编程 | 协议管理 |
| 位置 | 侧边栏/底部 | Webview 面板 |
| 版本显示 | 窗口标题 | 状态栏 |

## ❓ 故障排除

### 问题: 状态栏没有显示 ClaudeMD
**解决方案:**
1. 按 `Ctrl+Shift+P`
2. 输入: `Developer: Reload Window`
3. 重新加载窗口

### 问题: 找不到命令
**解决方案:**
1. 确认扩展已启用（扩展面板中检查）
2. 重新加载窗口
3. 查看输出面板是否有错误（`Ctrl+Shift+U`）

### 问题: 想卸载其他 Claude 扩展
如果您只想用我们的插件管理 CLAUDE.md：
1. 打开扩展面板
2. 找到 `lkbaba.claude-code-chatui`
3. 点击卸载（不会影响我们的插件）

## 📝 快速上手

安装确认后，建议按以下步骤开始使用：

1. **打开主面板**: 点击状态栏 `☁ ClaudeMD v2.1.1`
2. **更新协议**: 点击 "更新协议" 按钮
3. **应用到项目**: 点击 "应用协议" 按钮
4. **查看 Tips**: 点击 "Tips 管理" 按钮
