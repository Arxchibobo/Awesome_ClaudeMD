# ClaudeMD Manager - 手动清理安装指南

如果自动脚本失败，请按照以下步骤手动清理安装。

## 📋 完整步骤

### 步骤 1：卸载旧版本

在**命令提示符**或 **PowerShell** 中运行：

```bash
code --uninstall-extension claudemd-team.claudemd-manager
```

### 步骤 2：手动删除扩展目录

1. 按 `Win + R`，输入以下路径并回车：
   ```
   %USERPROFILE%\.vscode\extensions
   ```

2. 在打开的文件夹中，找到并删除所有名称以下开头的文件夹：
   ```
   claudemd-team.claudemd-manager-
   ```

### 步骤 3：关闭所有 VS Code 进程

**方法 A：使用任务管理器**
1. 按 `Ctrl + Shift + Esc` 打开任务管理器
2. 找到所有 "Code.exe" 进程
3. 右键 → 结束任务

**方法 B：使用命令行**
```bash
taskkill /F /IM Code.exe /T
```

### 步骤 4：清理工作区缓存（重要！）

这是 **最关键** 的一步，用于清除 webview 缓存。

1. 按 `Win + R`，输入以下路径并回车：
   ```
   %APPDATA%\Code\User\workspaceStorage
   ```

2. 在打开的文件夹中：
   - **选项 A（安全）：** 删除所有 1 天前的文件夹
   - **选项 B（彻底）：** 删除整个 `workspaceStorage` 文件夹（VS Code 会重新创建）

### 步骤 5：安装新版本

在 **PowerShell** 中运行：

```powershell
cd "E:\Bobo's Coding cache\Awesome_ClaudeMD\claude-code-chatinwindows"
code --install-extension claudemd-manager-1.0.0.vsix --force
```

### 步骤 6：完全重启 VS Code

**重要：不要使用 "Reload Window"，必须完全关闭再打开**

1. 确保所有 VS Code 窗口都已关闭
2. 检查任务管理器中没有 Code.exe 进程
3. 重新启动 VS Code

### 步骤 7：验证安装

1. 点击左侧 ClaudeMD 图标
2. 打开 "Tips 管理" 面板
3. 按 `F12` 打开开发者工具
4. 切换到 "Console" 标签

**✅ 预期结果（正确）：**
```
[TipsPanel] 开始更新数据...
[TipsPanel] 获取到 X 个待整合 Tips
[TipsPanel] ✅ 数据更新成功
```

**❌ 错误结果（如果看到这些，说明缓存未清理）：**
```
Uncaught SyntaxError: Failed to execute 'write' on 'Document'
Uncaught ReferenceError: switchTab is not defined
```

## 🔧 故障排除

### 如果仍然看到错误

1. **彻底删除工作区缓存：**
   ```powershell
   Remove-Item "$env:APPDATA\Code\User\workspaceStorage" -Recurse -Force
   ```

2. **清理 VS Code 所有扩展缓存：**
   ```powershell
   Remove-Item "$env:USERPROFILE\.vscode\extensions\claudemd-team.claudemd-manager-*" -Recurse -Force
   ```

3. **重启电脑（如果上述都不行）：**
   - 有时 Windows 文件锁定需要重启才能释放

### 验证文件版本

检查 VSIX 包是否是最新的：

```bash
cd "E:\Bobo's Coding cache\Awesome_ClaudeMD\claude-code-chatinwindows"
dir claudemd-manager-1.0.0.vsix
```

- 文件大小应该是：**约 1.19 MB (1,190,000 字节)**
- 如果小于 1 MB，说明是旧版本，需要重新打包

### 重新打包（如果需要）

```bash
cd "E:\Bobo's Coding cache\Awesome_ClaudeMD\claude-code-chatinwindows"
rm -f claudemd-manager-*.vsix
npm run compile
npx @vscode/vsce package
```
