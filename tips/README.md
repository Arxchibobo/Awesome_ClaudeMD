# Tips 贡献指南

团队 Claude Code 避坑经验收集。

## 贡献流程

1. 在 `tips/` 下新建 md 文件，命名：`<主题>-<姓名>.md`
2. 按模板填写（参考 `_template.md`）
3. 提交到 main 分支
4. GitHub Actions 自动整合到核心协议

## 模板

```markdown
# 主题名称

## 问题

简述遇到的问题

## 解决方案

Claude 应该如何处理

## 示例（可选）

具体例子
```
