# Tips 贡献指南

这里收集团队成员在使用 Claude Code 时发现的避坑经验。

## 如何贡献

1. 在 `tips/` 目录下创建你的 md 文件，命名格式：`<主题>.md`
2. 提交 PR 到 main 分支
3. 等待 review 合并

## 文件模板

```markdown
# 主题名称

## 问题描述

简述你遇到的问题

## 解决方案

Claude 应该如何处理

## 示例

（可选）给出具体例子
```
